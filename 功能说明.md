# 身份证识别系统 - 功能说明

## 🎉 新增功能

### 1. ✨ 美化界面
- **渐变色标题栏**：蓝色渐变设计，更加现代美观
- **卡片式布局**：使用圆角卡片，层次分明
- **图标按钮**：每个按钮都有对应的 Emoji 图标，更直观
- **色彩区分**：不同功能使用不同颜色，易于识别
  - 蓝色：选择图片
  - 绿色：识别操作
  - 橙色：查看结果
  - 紫色/粉色：批量操作

### 2. 🔄 身份证背面识别
- **正反面选择**：通过单选按钮选择识别正面或反面
- **正面信息**：姓名、性别、民族、出生日期、户籍地址、身份证号
- **背面信息**：签发机关、有效期限
- **智能显示**：根据识别内容自动显示/隐藏相关字段

### 3. 📚 批量识别处理
- **多图选择**：一次性选择多张身份证照片
- **批量识别**：自动依次识别所有选中的照片
- **进度提示**：实时显示当前识别进度（第 X/总数 张）
- **结果统计**：显示成功/失败数量
- **结果展示**：
  - 列表形式展示所有识别结果
  - 成功项显示姓名和身份证号
  - 失败项显示错误信息
- **数据导出**：
  - 导出为 CSV 格式
  - 包含所有识别字段
  - 自动生成时间戳文件名
  - 保存到应用文档目录

## 📱 使用说明

### 单张识别流程
1. 选择"正面（人像面）"或"反面（国徽面）"
2. 点击"📁 选择身份证照片"
3. 从相册选择一张身份证照片
4. 点击"🔍 开始识别"
5. 等待识别完成
6. 点击"📋 查看识别结果"

### 批量识别流程
1. 点击"📂 选择多张照片"
2. 在相册中选择多张身份证照片（长按可多选）
3. 确认选择，显示已选数量
4. 点击"⚡ 批量识别"
5. 等待批量识别完成
6. 点击"📊 查看批量结果"
7. 可选：点击"💾 导出数据"保存为 CSV 文件

## 🔧 技术实现

### 核心类
- `MainActivity.java`：主界面，处理单张和批量识别
- `ResultActivity.java`：单张识别结果展示
- `BatchResultActivity.java`：批量识别结果展示
- `BatchResultAdapter.java`：批量结果列表适配器
- `OCRUtil.java`：OCR 识别工具类（支持正反面）
- `IdCardInfo.java`：身份证信息数据模型
- `BatchResultManager.java`：批量结果管理器

### 数据流
```
选择图片 → OCR识别 → 解析JSON → 数据模型 → 界面展示 → 导出CSV
```

## 📝 导出格式

CSV 文件包含以下字段：
- 序号
- 姓名
- 性别
- 民族
- 出生日期
- 户籍地址
- 身份证号
- 签发机关
- 有效期限
- 状态（成功/失败）
- 错误信息

文件名格式：`身份证识别结果_20260415_143025.csv`

## ⚠️ 注意事项

1. **网络连接**：需要联网才能使用 OCR 识别功能
2. **照片质量**：照片清晰度影响识别准确率
3. **正反面选择**：识别前请正确选择正面或反面
4. **批量识别**：建议每次不超过 20 张，避免等待时间过长
5. **存储权限**：导出功能需要存储权限
